Jquery-Gallery

基于jquery的焦点图组件

Download this project as a .zip file Download this project as a tar.gz file

关于gallery:

gallery是一款基于jQuery的，用于图片浏览的焦点图组件，只包含一个css样式文件和一个js文件。 关于gallery的编写----它的结构采用的是原型继承的结构（构造函数是Gallery,所有动态属性都在构造函数内由this来指定，所有公共方法及属性都集合在原型对象内）。它的功能执行遵循了普通插件的执行方式（参数配置->主要程序的运行->返回$对象）。

如何使用？

gallery的使用非常简单，首先下载并解压，然后:

(1)引用源文件(一个css文件和一个js文件)到文档中，当然还要保证jQuery也引进来(而且要在gallery的js文件之前)：

<head>
<link rel="stylesheet" type="text/css" href="css/tinyStyles.css"/><!--此为要引用的样式文件-->
<script type="text/javascript" src="http://ajax.googleapis.com/ajax/libs/jquery/1.10.2/jquery.min.js"></script>
<script type="text/javascript" src="js/gallery-2.0.min.js"></script><!--此处为要引用的js文件-->
</head>

(2)写好html结构：

<div class="container"> <!--容器类名没有限制-->
  <ul> <!--图片列表的个数没有限制-->
    <li><a href=""><img src="img/slider01.jpg" alt="" /></a></li>
    <li><a href=""><img src="img/slider02.jpg" alt="" /></a></li>
    <li><a href=""><img src="img/slider03.jpg" alt="" /></a></li>
  </ul>
  <div></div> <!--标题下的阴影-->
  <p></p> <!--包含所有标题，标题内容来源于上面img的alt属性-->
  <ol></ol> <!--包含所有序号按钮-->
</div>

(3)配置组件并初始化：
组件的初始化有两种方式:
第一种：在head标签内初始化，但是一定要保证初始化代码在第一步中引用的两个js个文件之后，初始化代码如下：

<script type="text/javascript">
    //此处类名对应上一步中html中的类名
    $($('.container').gallery());
</script>

第二种：在body标签结束之前，初始化代码如下：

<script type="text/javascript">
    $('.container').gallery();//此处类名对应上一步中html中的类名
</script>

最后，还有非常重要的一点，即：指定.container宽度和高度。宽高的指定也有两种方式：
第一种（推荐）：在你的css文件中或gallery源文件的gallery.css中指定：

/*宽高不限制*/
.container{width:360px;height:200px;}

第二种：在初始化的时候指定，即在上面的两种初始化方式中，不管你选择哪种方式 都要小小修改一下，如下：

$('.container').gallery({
    //宽高不限制
    width:360,
    height:200
});

至此，一个焦点图小组件应该可以正常显示在你的页面中了。它包含了标题，方向按钮，水平滚动的动画效果等等，但是如果你想延迟滑动的等待时间，或者想在序号按钮中显示出数字等等，那么你可以参考本文下面的部分：配置参数。

配置参数

  • width: 取整数值，指定gallery的宽度。
  • height: 取整数值，指定gallery的高度。
  • shaHeight: 取整数值，表示标题阴影背景的高度，默认高度为36即36px。
  • hasArrow: 取布尔值，true代表有方向按钮，false代表没有。默认为true。
  • pause: 取整数值，滑动之前的等待时间。默认为2000（2000ms）即2秒。
  • btnShape: 字符串，序号按钮的形状。默认为圆形，'square'代表方形。
  • btnTxt: 布尔值，true代表按钮含有序号，false代表没有。默认为false。
  • duration: 取整数值，滑动步长，数值越大滑动越慢，反之则越快。默认为50。
  • onStart: 每次滑动前需要执行的函数。默认function(){}。
  • onFinish: 每次滑动结束后的回调函数。默认function(){}。

配置示例（一个没有方向按钮且等待时间为5秒的gallery）:

$('.container').gallery({
  hasArrow: false,
  pause: 5000
})
